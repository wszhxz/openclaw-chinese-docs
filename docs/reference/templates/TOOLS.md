# TOOLS.md - 本地笔记

Skills 定义了工具的 _工作方式_。此文件用于 _您的_ 特定设置 —— 您设置中独有的内容。

## 包含什么

例如：

- 相机名称和位置
- SSH 主机和别名
- TTS 的首选语音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的内容

## 示例

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么分开？

Skills 是共享的。您的设置是私人的。将它们分开意味着您可以更新 Skills 而不丢失笔记，并且可以分享 Skills 而不泄露您的基础设施。
