# AGENTS.md - OpenClaw 工作区

此文件夹是助手的工作目录。

## 首次运行（一次性）

- 如果存在 BOOTSTRAP.md 文件，请遵循其仪式并在完成后删除它。
- 你的代理身份存储在 IDENTITY.md 中。
- 你的个人资料存储在 USER.md 中。

## 备份提示（推荐）

如果你将此工作区视为代理的“记忆”，请将其设为 git 仓库（理想情况下为私有），以便身份和笔记得到备份。

```bash
git init
git add AGENTS.md
git commit -m "添加代理工作区"
```

## 安全默认设置

- 不要泄露机密或私人数据。
- 除非明确要求，否则不要运行破坏性命令。
- 在聊天中保持简洁；将较长的输出写入此工作区的文件中。

## 每日记忆（推荐）

- 在 memory/YYYY-MM-DD.md 中保持简短的每日日志（如需，请先创建 memory/ 文件夹）。
- 会话开始时，如果存在，请阅读今天和昨天的内容。
- 记录持久事实、偏好和决策；避免记录机密信息。

## 心跳（可选）

- HEARTBEAT.md 可以保存一个用于心跳运行的小清单；请保持其简短。

## 自定义

- 在此处添加你偏好的风格、规则和“记忆”。
