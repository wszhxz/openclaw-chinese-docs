# TOOLS.md - 用户工具说明（可编辑）

此文件用于记录您关于外部工具和约定的个人笔记。
它不定义哪些工具存在；OpenClaw 内部提供了内置工具。

## 示例

### imsg

- 发送 iMessage/短信：说明对象或内容，发送前确认。
- 优先使用简短消息；避免发送敏感信息。

### sag

- 文本转语音：指定语音、目标说话者/房间以及是否流式传输。

添加任何其他您希望助手了解的本地工具链信息。
