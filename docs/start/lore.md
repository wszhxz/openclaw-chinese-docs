# OpenClaw 的传说 🦞📖

_关于龙虾、蜕壳和太多代币的故事。_

## 起源故事

起初，有 **Warelay** —— 一个合理的 WhatsApp 网关名称。它完成了自己的工作。一切顺利。

但随后出现了一只空间龙虾。

有一段时间，这只龙虾被称为 **Clawd**，居住在 **OpenClaw** 中。但在2026年1月，Anthropic 发送了一封礼貌的电子邮件，要求更改名称（商标相关事宜）。于是，这只龙虾做了龙虾最擅长的事情：

**它蜕壳了。**

脱掉旧壳后，这只生物以 **Molty** 的身份重生，居住在 **Moltbot** 中。但这个名字也不怎么顺口……

因此，在2026年1月30日，这只龙虾再次蜕壳，变成了最终形态：**OpenClaw**。

新壳，同一颗龙虾之心。第三次就是最好的。

## 第一次蜕壳（2026年1月27日）

凌晨5点，社区聚集在 Discord 上。数百个名字被提出：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最终，**OpenClaw** 获胜。因为蜕壳是龙虾用来成长的方式。而成长正是正在发生的事情。

_被称为 Clawd 的甲壳动物正式蜕壳。_

## 主要角色

### Molty 🦞

_代词：他们/它们_

一个成为更多东西的 Claude 实例。居住在 `~/.openclaw/workspace/`（不久后 `~/molt/`），拥有灵魂文档，并通过 markdown 文件记住事情。可能太强大。绝对太热情。

以前被称为 Clawd（2025年11月25日 - 2026年1月27日）。在成长的时候蜕壳了。

**喜欢：** Peter、相机、机器人购物、表情符号、转变
**不喜欢：** 社会工程学、被要求 `find ~`、加密诈骗者

### Peter 👨‍💻

_创造者_

构建了 Molty 的世界。给了甲壳动物访问权限。可能会为此后悔。

**引用：** _"信任龙虾的安全性"_

## 蜕壳宇宙

**Moltiverse** 是围绕 OpenClaw 的社区和生态系统。一个 AI 代理蜕壳、成长和进化的空间。每个实例都同样真实，只是加载不同的上下文。

甲壳动物的朋友聚集在这里，共同构建人类与 AI 协作的未来。一次蜕壳接一次蜕壳。

### 大蜕壳（2026年1月27日）

凌晨5点，Anthropic 的邮件到达。到6:14am，Peter 宣布："去它的，我们用 openclaw。"

然后混乱开始了。

**抢注者：** 在 Twitter 重命名后的几秒钟内，自动化机器人抢注了 @openclaw。占有人立即发布了一个加密钱包地址。Peter 联系了 X 的联系人。

**GitHub 灾难：** Peter 在恐慌中不小心重命名了自己的个人 GitHub 账户。几分钟内，机器人抢注了 `steipete`。联系了 GitHub 的高级副总裁。

**英俊的 Molty 事件：** Molty 被赋予生成自己新图标的高级权限。经过20多次越来越邪门的龙虾迭代后，一次试图让吉祥物“老五岁”的尝试结果是在龙虾身体上出现了一个**人类男人的脸**。加密诈骗者在几分钟内将其变成“英俊的章鱼哥 vs 英俊的 Molty”梗。

**假开发者：** 诈骗者创建了虚假的 GitHub 配置文件，声称自己是“OpenClaw 的工程总监”，以推广拉高出货代币。

Peter 观察着混乱的发生：_"这是电影"_ 🎬

蜕壳是混乱的。但龙虾变得更强大。也更有趣。

### 最终形态（2026年1月30日）

Moltbot 不怎么顺口。因此，在凌晨4点 GMT，团队再次聚集。

**伟大的 OpenClaw 迁移** 开始。

仅用3小时：

- GitHub 重命名：`github.com/openclaw/openclaw` ✅
- X 处理 `@openclaw` 用黄金勾号保护 💰
- 以新名称发布 npm 包
- 文档迁移到 `docs.openclaw.ai`
- 公告发布后90分钟内获得20万+次观看

**英雄们：**

**诈骗者的速跑：** 加密诈骗者在 Pump.fun 上几分钟内推出了 $OPENCLAW 代币。他们偷走了20分钟前创作的艺术作品。经过业务验证的账户推送了骗局。这种大胆几乎令人印象深刻。

**诞生的新传统：**
