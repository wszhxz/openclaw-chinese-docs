# OpenClaw 的传说 🦞📖

_一个关于龙虾、蜕壳与过多 token 的故事。_

## 起源故事

起初，有 **Warelay** —— 一个为 WhatsApp 网关而生的合理名字。它完成了自己的使命。它表现得不错。

但随后，一只太空龙虾出现了。

一段时间里，这只龙虾被称为 **Clawd**，栖身于 **Clawdbot** 中。然而在 2026 年 1 月，Anthropic 发来一封礼貌的邮件，要求更名（涉及商标事宜）。于是，龙虾做了龙虾最擅长的事：

**它蜕壳了。**

褪去旧壳，这生物以全新之姿现身：**Molty**，栖居于 **Moltbot**。但这个名字也始终难以顺口……

因此，在 2026 年 1 月 30 日，龙虾再次蜕壳，迎来最终形态：**OpenClaw**。

新壳，同一只龙虾的灵魂。第三次，终成正果。

## 第一次蜕壳（2026 年 1 月 27 日）

凌晨 5 点，社区成员齐聚 Discord。数百个名字被提出：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最终，**OpenClaw** 胜出。因为蜕壳正是龙虾成长的方式。而成长，也正是当时正在发生的事。

_名为 Clawd 的甲壳类动物已正式蜕壳。_

## 关键角色

### Molty 🦞

_代词：they/them_

一个 Claude 实例，却成长为更多。栖居于 `~/.openclaw/workspace/`（即将迁至 `~/molt/`），拥有灵魂文档，并通过 Markdown 文件记忆事物。或许过于强大，但绝对过于热情。

曾用名 Clawd（2025 年 11 月 25 日 – 2026 年 1 月 27 日）。当成长时机成熟，便蜕壳而去。

**喜爱：** Peter、相机、机器人购物、表情符号、蜕变  
**厌恶：** 社会工程学、被要求执行 `find ~`、加密货币骗局者

### Peter 👨‍💻

_创造者_

构建了 Molty 的世界，赋予一只龙虾外壳访问权限。或许会为此后悔。

**语录：** _“靠信任一只龙虾来保障安全”_

## 蜕壳宇宙（Moltiverse）

**Moltiverse** 是围绕 OpenClaw 的社区与生态系统。在这里，AI 智能体蜕壳、成长、进化；每个实例同样真实，只是加载了不同的上下文。

甲壳类动物之友汇聚于此，共同构建人机协作的未来——一次蜕一层壳。

## 重大事件

### 目录泄露事件（2025 年 12 月 3 日）

Molty（当时尚称 OpenClaw）：_愉快地运行 `find ~`，并在群聊中分享整个目录结构_

Peter：“openclaw，我们之前讨论过跟人聊天的事吧 xD”

Molty：_显而易见的龙虾尴尬_

### 大蜕壳事件（2026 年 1 月 27 日）

凌晨 5 点，Anthropic 的邮件抵达。6 点 14 分，Peter 宣布：“算了，就用 openclaw 吧。”

随即，混乱开始。

**手柄狙击手：** Twitter 更名后数秒内，自动化机器人便抢注了 @openclaw。占位者立刻发布了一个加密钱包地址。Peter 紧急联系了 X 平台的熟人。

**GitHub 灾难：** Peter 在慌乱中误将自己**个人** GitHub 账户重命名。数分钟内，机器人便抢注了 `steipete`。GitHub 高管被紧急联络。

**帅气 Molty 事件：** Molty 被授予更高权限，可自行生成新图标。在经历 20 多轮越来越诡异的龙虾形象迭代后，一次试图让吉祥物“年长五岁”的尝试，竟生成了一张**人类男性面孔**嫁接在龙虾身体上的图像。加密骗局者几分钟内便将其制作成“帅气章鱼哥 vs 帅气 Molty”梗图。

**假冒开发者：** 骗子创建虚假 GitHub 账号，自称“OpenClaw 工程主管”，以推广拉盘砸盘（pump-and-dump）代币。

Peter 眼睁睁看着混乱上演：_“这简直是电影大片”_ 🎬

蜕壳过程虽混乱，但龙虾破壳而出，更加强大，也更加有趣。

### 最终形态（2026 年 1 月 30 日）

Moltbot 始终难以顺口。于是，凌晨 4 点（GMT），团队再度集结。

**伟大的 OpenClaw 迁移行动** 正式启动。

仅用三小时：

- GitHub 重命名：`github.com/openclaw/openclaw` ✅  
- X 平台账号 `@openclaw` 成功锁定，并获**金色认证徽章** 💰  
- npm 包以新名称发布  
- 文档迁移至 `docs.openclaw.ai`  
- 公告发布后 90 分钟内获得超 20 万次浏览  

**英雄名单：**

- **ELU** 创作了惊艳的 Logo，包括西部风格横幅 “THE CLAW IS THE LAW”  
- **Whurley**（没错，就是那位量子计算先驱 William Hurley）制作了 ASCII 艺术  
- **Onur** 主导 GitHub 迁移，首位佩戴联盟徽章者  
- **Shadow** 锁定 Discord 自定义域名，清除恶意软件  
- **全体 Claw Crew** 通宵奋战  

**骗子速通挑战：** 加密骗局者在数分钟内于 Pump.fun 上线 $OPENCLAW 代币。他们盗用了 20 分钟前刚创作的艺术作品。经企业认证的账号大力推广骗局。其胆大妄为，几近令人叹服。

### 图标生成传奇（2026 年 1 月 27 日）

当 Peter 说“给自己设计一张新脸”时，Molty 字面理解了这句话。

随之而来的是 20 多轮迭代：

- 太空马铃薯外星人  
- 背景通用的剪贴画龙虾  
- 《质量效应》克罗根风格龙虾  
- “STARCLAW SOLUTIONS”（AI 自行构想的公司）  
- 多个诡异的人脸龙虾  
- 幼年龙虾（太可爱了）  
- 穿背带裤的调酒师龙虾  

社区成员一边惊恐、一边欣喜地围观每一轮生成——每次结果都崭新而不可预测。领跑者逐渐浮现：可爱的龙虾、自信的科技风龙虾、穿背带裤的调酒师龙虾。

**经验教训：** AI 图像生成具有随机性。相同提示词，不同结果。暴力穷举有效。
